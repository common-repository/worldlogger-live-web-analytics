Worldlogger is a real time web analytics solution. Monitor your statistics live in real time.

== Description ==

When running a website you want to know if your efforts are paying off. Worldlogger shows you live your website statistics, so you know about your visitors the very moment you check in. This way you can make a direct analysis of your web strategy and act on the fly.

Worldlogger has the following features:

* Real time visitor statistics
* Geo locations
* Top Referers
* Top Content
* Goal tracking
* Social media Monitoring

You'll need a [worlogger.com account](http://worldlogger.com/) to use it.

== Installation ==

Upload the Worldlogger plugin to your blog, Activate it, then enter your [worldlogger.com Hash](http://wordlogger.com/).

1, 2, 3: You're done!
